おすすめ

クラゲ館

クラゲの美しさに魅せられて常時50種以上のクラゲを展示し、その生態や魅力を伝えます。
クラゲが創りだす特別なひと時をお楽しみください。

イルカショー

水族館のスターたちのショーは「わくわく」・「ドキドキ」。
トレーナーと織りなす華麗なエンターテインメントをお楽しみください。

(開催時間: 11:00～ 、13:00～ 、15:00～ )

ナイトツアー

ライトアップされた館内を散策。
水槽の光が反射してなんともロマンティック。
ガイド付きのツアーも実施中。普段では見られない生き物たちの寝顔や生態を観察できます。

(4月～10月: 19:30 ～ 21:00
/ 11月～3月: 18:30 ～ 20:00)

イベント

イルカと遊ぼう

水族館の人気者”イルカ”と直接ふれあえる。
トレーナーになった気分でイルカにエサをあげたり、ジャンプなどの指示を出したり。
さらに、水に入ってイルカと一緒に泳げる体験も。
優しい目をした愛嬌たっぷりのイルカに心から癒されます。
小さなお子様も参加できるので、ご家族でご参加いただけます。

小魚プール

小魚と遊べる水遊びプール。
元気いっぱい泳ぎ回る小魚たちと、追いかけっこをしたり、触ってみたり。
ご家族でお楽しみいただけます。タオルや着替えをご用意下さい。
※ 足のみの入水にしてください。
※ 期間限定: 4月～10月のみ開催

館内のなかまたち

イルカ

20頭から、多いときは約100頭の群れで生活し、人間には聞こえない音で仲間同士コミュニケーションをとります。 一生を海の中で生活するけど、わたしたち人間と同じ哺乳類です。

体長 : 1.8～3.8m
体重 : 200～240kg
住みか : 世界中の熱帯～温帯地域の海
食べ物 : 小魚、いか など

アシカ

日本の動物園や水族館で多く飼育されているアシカ。オスはメスよりも体が大きく、成長すると頭のてっぺんが盛り上がってきます。好奇心旺盛で、頭も良いのでショーで活躍しています。

体長 : 2.0～2.4m
体重 : 90～390kg
住みか : 太平洋東部のカナダからメキシコにかけて
食べ物 : 小魚、いか、たこ など

オットセイ

アシカの仲間ですが、アシカよりも毛深く、短い毛と長い毛が二重に生えています。 質の高い毛皮ができることから、かつて乱獲され数を減らしてしまいました。

体長 : 1.5～2.1m
体重 : 50～270kg
住みか : 太平洋北部、ベーリング海、オホーツク海
食べ物 : 小魚、いか、たこ など

アザラシ

寒い海に生息するアザラシで、2月から4月にかけて流氷の上で赤ちゃんを産みます。 お母さんアザラシはたった2週間ほどしか赤ちゃんの面倒をみず、赤ちゃんはすぐにひとり立ちします。

体長 : 1.5～1.7m
体重 : 70～120kg
住みか : 北太平洋、日本海、オホーツク海、ベーリング海
食べ物 : 海底のエビやカニの仲間、小魚、いか、たこ など

フンボルトペンギン

南米チリ～ペルー沿岸の温帯域に住むペンギン。ペンギンと聞くと南極にいると思われがちですが、フンボルトペンギンはサボテンの生える乾燥地域に暮らしています。

体長 : 約70cm
体重 : 約4kg
住みか : ペルー、チリ沿岸海域
食べ物 : イワシ類

セイウチ

鼻の周りには約500本のテニスラケットのガットのようなヒゲが生え、人間の指のように繊細な感覚を持っています。このヒゲを使って海底に埋もれた貝を探し当てて食べます。

体長 : 2.5～3.5m
体重 : 550～1900kg
住みか : 北極海とその周辺の海
食べ物 : 二枚貝、いか、ゴカイ、巻貝、動きの遅い魚 など
